# Сценарий 2.9: Оформление заказа

**Цель:** быстро и просто оформить заказ с позициями, размещенными в Корзине

## Основной сценарий:

1. Пользователь открывает Корзину
2. Система отображает содержимое Корзины и контактные данные пользователя, заполненные автоматически из Личного кабинета (ФИО, Телефон, Email, Адрес доставки)
3. Система автоматически заполняет параметры вручения заказа и способ оплаты значениями из предыдущего заказа
4. Пользователь вводит промокод или купон при их наличии
5. Система пересчитывает итоговую сумму с учетом введенного промокода или купона
6. Пользователь проверяет содержимое заказа и корректность контактных данных
7. **Кнопка «Подтвердить заказ» становится доступной для нажатия**
8. Пользователь нажимает кнопку «Оформить»
9. Система проверяет наличие всех позиций и топпингов
10. Система сохраняет заказ и присваивает ему статус «Оформлен»
